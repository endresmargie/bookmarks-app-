# Bookmarks App

A lightweight, no-server bookmark manager built with vanilla HTML, CSS, and JavaScript.

## Features

- Add bookmarks with a title, URL, and category
- Bookmarks saved to `localStorage` — persists across page refreshes
- Bookmarks grouped by category
- Search/filter bookmarks by title in real time
- Edit bookmarks inline
- Delete bookmarks
- Dark mode toggle (remembers your preference)
- Contact form with inline validation

## Project Structure

```
index.html       # Main landing page with nav, hero, about cards, and contact form
bookmarks.html   # Bookmark manager app
contact.html     # Standalone contact form
dark.css         # Dark mode stylesheet
```

## How to Use

No build steps or dependencies required. Just open `index.html` in any browser.

```bash
open index.html
```

## Built With

- HTML, CSS, JavaScript
- `localStorage` for data persistence
- Google Favicon API for bookmark icons
